Bienvenue sur le dépôt officiel de paquets MythicOS

Ajouter le dépôt

Ce dépôt est déjà intégré dans MythicOS. Pour les autres distributions :

Clé :

wget -O - https://packages.mythicos.hastag.fr/mythicos-key.asc | sudo tee /etc/apt/keyrings/mythicos.gpg

Dépôt :

deb [signed-by=/etc/apt/keyrings/mythicos.gpg] https://packages.mythicos.hastag.fr stable main
